JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremy popular of a variety of reasons. It is a small and lightweight language allwoing maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host enviornment (a web browser or Node server), it can be connected to the objects of these enviornments to provide programmatic control over them.

  • Variable Declaration JavaScript variles are containers for storing data values - imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a value. All JavaScript varibles must be identified with unique names. These unique names are called identifiers var x;
  • Variable Assignment Assignment operators assign values to JavaScript variables - our cup can now have coffee poured in it, fiving our variable a value to hold. The = assignment operator assigns a value to a variable. var x=10;
  • Data Types Data types are an important concept; to be able to operate on variables you need to know the data type. Ther are six data types that are JavaScript primitives: Boolean - True or false; null -null aka nothing; Number - 42 or 3.14159; String- "Coding Dojo Rocks!" Array -[1,'Coding',2,'Dojo']; and Object - {first_name: 'Jane', last_name: 'Doe'}
  • Conditionals When you write code, you want to perform differnet actions for different decisions - hitting differnet code blocks based on values or conditionals that have been met. You can use conditional statements in your code to accomplish this. There are the following conditional statements: if a specified condition is true, do this doce block; else if to specify a new condition to test, if the firs condition is flase; else we execute this block of code;
  • Loopsthre are many differnt kinds of loops in every programming language, but they all essentially do the same thing: they will repeat an action some number of times. Imagine you ahe to run a mile, well you run around the track four times and then you stop. That's a loop!
  • Function Functions are an encapulsation of a code block. Think of it as a list of instructions. As an example imagine we ar putting togather a desk from Ikea, we open up the instruction manual and get started, first we screw the legs to the table top; next we plave the table the right way up.
